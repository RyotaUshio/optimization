SWIGによる外部関数インターフェース(FFI)
=================

C++で書かれた本ライブラリをPythonから呼び出すには、SWIGを利用する。

ビルド方法
-----
```
$ make
```

インポート
---
```
$ python
Python 2.7.17 (default, Sep 30 2020, 13:38:04) 
[GCC 7.5.0] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> import continuous
```
